# 📌 Descripción

Radar es un motor de control de riesgos en tiempo real diseñado principalmente para aplicaciones antifraude y plataformas financieras.

El sistema permite:

- Detectar riesgos en tiempo real
- Gestionar reglas dinámicamente
- Analizar eventos complejos
- Centralizar estrategias antifraude
- Automatizar respuestas de riesgo
- Configurar reglas mediante interfaces visuales

Todo esto con una arquitectura ligera, extensible y lista para usar.

# 🚀 Instalación

## 1️⃣ Clonar el repositorio

```bash
git clone https://github.com/wfh45678/radar.git
```

---

## 2️⃣ Configurar dependencias

Instalar:

- Java JDK 8+
- Maven
- MongoDB
- MySQL
- Redis
- Elasticsearch

---

## 3️⃣ Ejecutar el proyecto

```bash
mvn clean install
```

```bash
mvn spring-boot:run
```
